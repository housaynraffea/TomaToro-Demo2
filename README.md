This is a short, but simple demonstration of a pomodoro timer app made in Android Studio. Not all features are implemented, but only the important ones.

Likewise, from the first demonstration repository, as soon as you press 'START', the program will pick up what number you wrote in the 'Study', 'Short Break', and 'Long Break' entries and create a timer. It will then display a popup when the timer has run out of time.

This app also uses the same method of ending a session, as in displaying an alert popup instead of a notification in the background.
